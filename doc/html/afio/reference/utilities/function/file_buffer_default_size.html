file_buffer_default_size

Returns a reasonable default size for file_buffer_allocator, typically the closest page size from page_sizes() to 1Mb.

Synopsis 
size_t utils::file_buffer_default_size()
Parameters

Type

Concept

Name

Description

Returns

A value of a TLB large page size close to 1Mb.

Header

#include <boost/afio/afio.hpp>

Complexity

Whatever the system API takes (one would hope constant time).

Exception Model

Any error from the operating system or std::bad_alloc.
